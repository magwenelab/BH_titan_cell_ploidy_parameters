This repository contains scripts and parameters for analysis of diploid titan cell progeny. 

WeavePop_parameters: Files used for recreating the results of the WeavePop pipeline on the BH progeny.
* config.yaml - Configuration file for WeavePop with parameters
* chromosomes.csv and loci.csv - Input files for WeavePop used alongside sequences and sample names (not included in this repository)

simulated_reads: Script used to simulate long reads with recombination events from reference genomes.
